# Cool Zone Expert - AC Service Website

A professional website for Cool Zone Expert, your trusted air conditioning service provider.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional design with smooth animations
- **Service Showcase**: Highlights all AC services including installation, repair, maintenance, and emergency service
- **Contact Form**: Easy-to-use quote request form for customer inquiries
- **Smooth Navigation**: Intuitive navigation with smooth scrolling

## Services Offered

- AC Installation
- AC Repair
- AC Maintenance
- 24/7 Emergency Service

## Getting Started

### Viewing the Website

Simply open `index.html` in your web browser to view the website locally.

### Deployment

You can deploy this website to any web hosting service:

1. **GitHub Pages**: Push to a GitHub repository and enable GitHub Pages
2. **Netlify**: Drag and drop the folder to Netlify
3. **Vercel**: Deploy directly from GitHub
4. **Traditional Hosting**: Upload files via FTP to your web server

## File Structure

```
coolzoneexpert/
├── index.html      # Main HTML file
├── styles.css      # CSS stylesheet
├── script.js       # JavaScript for interactivity
└── README.md       # This file
```

## Customization

- **Contact Information**: Update phone, email, and address in `index.html`
- **Colors**: Modify the color scheme in `styles.css`
- **Services**: Add or modify services in the services section
- **Form Handler**: Implement backend form handling in `script.js`
